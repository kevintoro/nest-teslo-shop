# Teslo Shop API

## 🚀 Quick Start

1. **Install NestJS CLI (if you don't have it yet)**
   ```bash
   npm install -g @nestjs/cli
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Configure Environment**
   - Copy `.env.example` to `.env` and edit as needed:
     ```bash
     cp .env.example .env
     ```
4. **Start Postgres via Docker**  
   _Requires [Docker](https://www.docker.com/) installed._
   ```bash
   docker-compose up -d
   ```
5. **Run the app**
   ```bash
   npm run dev
   ```
